NAME
       man - an interface to the on-line reference manuals

SYNOPSIS
       man page

DESCRIPTION
       man is the system's manual pager.  The page argument given to  man  is
       the  name of a function. The manual page associated with each of these 
	  arguments is then found and displayed. 

EXAMPLES
       man pwd
           Display the manual page for the item (program) pwd.
